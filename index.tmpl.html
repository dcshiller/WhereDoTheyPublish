About

This project is motivated by the thought that rankings should take into account the user's own interests and values. There is no adequate one-size-fits all ranking system. This project attempts to implement a flexible ranking system. Input your tastes, as measured by people who's work you admire, and the system outputs a list of journals based on those tastes.

The ranking provided here is not intended in any way to be a ranking of journal prestige, but instead to reflect something about which journals best reflect the interests and values of the input authors. Journals that are listed first are journals that include more work from the selected authors.

The project is on-going. At this stage, the search feature ranks journals based solely on the number of papers published by the input authors. It searchs for them in the PhilPapers database, and counts the number of publications per journal.

All data comes from PhilPapers. None is stored on this website. All code for this project can be found at my Github page.
